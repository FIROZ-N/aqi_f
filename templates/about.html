About AirQuality AI

Using cutting-edge machine learning technology, we transform complex environmental data into actionable insights that everyone can understand and use.

Machine Learning Model

Gradient Boosting Regressor

Our model uses an ensemble learning method that builds multiple decision trees sequentially, with each tree correcting the errors of the previous one.

Algorithm: Gradient Boosting Machine (GBM)
Estimators: 300 trees
Learning Rate: 0.05
Max Depth: 3 levels
Subsample: 80% of data
Features Used:
  • PM2.5 - Fine particulate matter
  • PM10 - Coarse particulate matter
  • NO₂ - Nitrogen dioxide
  • CO - Carbon monoxide
  • O₃ - Ozone
